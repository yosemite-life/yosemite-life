# Yosemite Life

Guided backpacking trips and private day hikes in Yosemite National Park. Small groups, permits handled, all meals included on backpacking trips. One guide on every trip.

Operated by [MountainsCalling.me LLC](https://mountainscalling.me) under a [National Park Service Commercial Use Authorization](https://www.nps.gov/yose/planyourvisit/authorizedguides.htm).

→ **[yosemite.life](https://yosemite.life)**

- [Guided Backpacking Trips](https://yosemite.life/backpacking/): Multi-day wilderness trips through Yosemite's high country
